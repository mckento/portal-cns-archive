# portal-cns-archive

## 项目简介

portal-cns-archive 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何单一域名或具体网站，而是作为一组静态页面的集中存放和版本管理空间。所有页面均以原始 HTML 文件形式存储，便于直接访问、审查和长期保存。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 仓库首页 / 页面索引（可选）
├── pages/             # 存放所有独立 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/            # 共享资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：主要页面归档目录，每个文件为一个独立的 HTML 页面。
- `assets/`：存放页面间共享的样式、脚本或图像资源，避免重复。
- `index.html`：可选的索引页，用于展示页面列表或导航。

## 页面归档说明

- 每个页面均为完整的独立 HTML 文件，包含必要的 HTML、CSS 和 JavaScript 代码。
- 页面之间没有强依赖关系，可单独打开或部署。
- 归档内容可能来自不同时期、不同用途的页面，仅作存档和展示之用。
- 本仓库不保证所有页面的时效性或外部链接的可用性。

## 维护说明

- 欢迎提交 Pull Request 新增或更新页面，但请确保页面内容符合开源社区规范。
- 新增页面请放入 `pages/` 目录，并遵循命名约定（如 `page-xxx.html`）。
- 如需更新现有页面，请说明修改原因。
- 本仓库由维护者定期审查，不活跃页面可能会被标记或移除。
- 如有任何问题，请通过 Issue 提出。
